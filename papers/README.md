# Company Brain Technical Report

## Company Brain: Human-Approved Operational Memory for Enterprise AI Agents

This folder contains the technical report behind Company Brain.

The paper argues that enterprise AI agents do not only need more retrieved context. Before agents take actions inside companies, they need source-backed, human-approved operating instructions that define what they are allowed to do, when to escalate, and what evidence supports the workflow.
